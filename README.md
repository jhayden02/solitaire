# solitaire

A solitaire implementation in C using raylib.

See [BUILDING.md](BUILDING.md) for complete instructions on how to set up your development environment.

To play the most recent commit of the game hosted on GitHub Pages, go [here](https://jhayden02.github.io/solitaire/).
